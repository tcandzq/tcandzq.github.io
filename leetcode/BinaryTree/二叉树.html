递推模板

迭代:使用栈来完成,我们先将根节点放入栈中,然后将其弹出,依次将该弹出的节点的右节点,和左节点,注意顺序,是右,左,为什么?因为栈是先入后出的,我们要后输出右节点,所以让它先进栈。

def preorderTraversal(root: TreeNode) -> List[int]:
	res = []
	if not root:
	    return res
    stack = [root]
    while stack:
        node = stack.pop()
        res.append(node.val)
        if node.right:
            stack.append(node.right)
        if node.left:
            stack.append(node.left)
    return res

def inorderTraversal(root: TreeNode) -> List[int]:
	res = []
	stack = []
	# 用p当做指针
	p = root
	while p or stack:
	    # 把左子树压入栈中
	    while p:
	        stack.append(p)
	        p = p.left
	    # 输出 栈顶元素
	    p = stack.pop()
	    res.append(p.val)
	    # 看右子树
	    p = p.right
	return res

# 层次遍历

经典的BFS实现

模板1:

def levelOrder(self, root):
	"""
	 :type root: TreeNode
	 :rtype: List[List[int]]
	 """
	 if not root:
	     return []
	
	 res,cur_level = [],[root]
	 while cur_level:
	     temp = []
	     next_level = []
	     for i in cur_level:
	         temp.append(i.val)
	
	         if i.left:
	             next_level.append(i.left)
	         if i.right:
	             next_level.append(i.right)
	     res.append(temp)
	     cur_level = next_level
	 return res

模板2:

使用python中的双端队列deque实现

from collections import deque

def levelOrder(root: TreeNode) -> List[List[int]]:
     if not root:
         return []
         
     queue = deque()
     queue.append(root)
     nums = []
     while queue:
         tmp = []
         for _ in range(len(queue)):
             node = queue.popleft()
             tmp.append(node.val)
             if node.left: queue.append(node.left)
             if node.right: queue.append(node.right)
         nums.append(tmp)
         queue = queue
     return nums

# 使用数组存储二叉树所有路径的权值

给定如下一颗二叉树

							              5
							             / \
							            4   8
							           /   / \
							          11  13  4
							         /  \    / \ 
							        7    2  5   1

问题1:使用数组存储根节点到叶子节点的所有路径如[[5,4,11,7],[5,4,11,2],[5,8,13],[5,8,4,1],[5,8,4,5]];

回答1：经典的DFS算法。需要在递归过程中保存中间参数。

递归函数的入参可以参考二叉树的前序遍历 你想 在二叉树的遍历过程 函数的入参是root 它在出栈后 不断访问 之前访问过的节点。所以像root那样 递归入参可以带你两次去到树的任何一个除了叶子节点的节点。

def pathSave(root:TreeNode) -> List[List[int]]:
	if not root:
		return []
	res = []
	def helper(root,tmp):
		if not root:
			return  #  递归的边界条件,一定要有
		if not root.left and not root.right: # 判断是否是叶子节点
			tmp += [root.val] #  这里很关键,因为递归函数走到这里入参是root.left和root.right，此时另外一个入参tmp+[tmp]追加的是叶子节点的父节点的值,所以需要把叶子节点的val加入到tmp中去
			res.append(tmp)
			return 
		helper(root.left,tmp+[root.val])
		helper(root.right,tmp+[root.val])
	helper(root,[])
	return res

问题2:使用数组存储以根节点开始，任意节点结束的所有路径如[[5],[5,4],[5,4,11],[5,4,11,7],[4,11],[4,11,7],[11,7],[7]]等;

回答2：双递归+前序遍历+递归过程中保存中间参数。

def pathSave(root:TreeNode):
	if not root:
		return []
	return paths(root) + pathSave(root.left) + pathSave(root.right)

def paths(root:TreeNode):
	if not root:
		return []
	res = []
	def helper(root:TreeNode,tmp):
		if not root:
			return
		if tmp and tmp not in res:
			res.append(tmp)
			if not root.left and not root.right:  # 叶子节点要额外处理
				res.append(tmp+[root.val])
				res.append([root.val])  # 单独把叶子节点append进去
		helper(root.left, tmp + [root.val])
		helper(root.right, tmp + [root.val])
	helper(root, [])
	return res
